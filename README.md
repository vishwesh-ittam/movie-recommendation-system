# 🎬 Movie Recommendation System

A content-based Movie Recommendation System developed using **Python**, **Pandas**, and **Scikit-learn**. The system recommends movies similar to a selected movie by computing feature similarity using the **Cosine Similarity** algorithm.

---

## 📌 Overview

Finding movies that match a user's interests can be difficult when thousands of options are available. This project implements a **content-based recommendation system** that suggests similar movies based on their metadata and feature similarity rather than user ratings.

---

## ✨ Features

- Content-based movie recommendations
- Data preprocessing using Pandas
- Similarity computation using Cosine Similarity
- Fast movie recommendation generation
- Simple and interactive notebook implementation

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Cosine Similarity
- Jupyter Notebook

---

## ⚙️ Workflow

1. Load the movie dataset.
2. Preprocess movie metadata.
3. Convert features into numerical representations.
4. Compute similarity using Cosine Similarity.
5. Recommend the most similar movies based on user input.
